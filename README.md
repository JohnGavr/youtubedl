## YTD - YoutubeDownloadBashScript

### Depedencies

1. youtube-dl 

Debian/Ubuntu: ```sudo apt install youtube-dl```

Arch based : ```sudo pacman -S youtube-dl```

### Usage Information

Basic usage: 

```./ytd --[option] [url]```

Available Options

1. mp3 : for mp3 convert
2. mp4 : for mp4 conver
3. help : for help options
